Depois de estudar, conhecer, usar e abusar do CSS você vai se perguntar: "Como a humanidade viveu tanto tempo sem CSS ?" Pois é uma ideia absurdamente simples, genial e que tornou os webmasters bem mais eficientes. O CSS poupa tempo, nos torna mais flexíveis. Nos permite fazer coisas que levariam horas e horas de trabalho manual (e isso de fato acontecia, até bem pouco tempo atrás).

O CSS nada mais é que uma folha de estilo. Um documento que vai conter todos os detalhes, declarações e código que vão definir como vai ser o estilo, o style de nosso site. É como se ele fosse um documento que dissesse: "Hey, navegador, exiba o título das postagens com tamanho 20px e cor azul na home do site. Caso o usuário clique no tutorial, o nome do título vai ter 25px e cor azul mais escura." "Os links vão sempre ser azuis, mas se ele clicar, vai ser tornar vermelho ok? Ah, sempre que ele passar o mouse sobre os botões, mude de cor." "Browser, dê um espaçamento maior entre os parágrafos. Um espaçamento de dois enters A largura do artigo é de 600px. Claro, se ele estiver no desktop. Se for um usuário mobile, diminua para 100px."
